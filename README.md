## ⚙️ Setup Instructions

### 1. Copy `.env.example` File

```
mv .env.example .env
```

### 2. Create API Keys

- Clerk - https://clerk.com/
- UploadThing - https://uploadthing.com/
- Neon DB - https://neon.com/

### 3. Install the dependencies

```bash
npm install
```
### 4. Run the App

```bash
npm run dev
```
